// What is a URL?

// What is DNS?
Domain Name Server - the phonebook/address book like the white pages. Helps locate the IP location from the URL 


// What is a query string?

// What are two HTTP verbs and how are they different?
GET and POST - GET asks for information from the server but doesn't change anything. POST adds to the server/database
